Frequently Asked Questions

How is it possible to determine if something is AI-generated?

When a Language Model writes code or text, it uses all its pre-training data to make a response based on statistics. This is significantly different from how humans write. To our AI detector, the presence of an AI generator becomes clear when compared to a large collection of human writing and coding patterns.

What models can you detect?

We can detect code and text generated by all major AI models including ChatGPT, Gemini, Claude, and others. Our system is continuously updated as new models are released.

How was the AI detection model trained?

Our model was trained on a vast dataset of both human-written and AI-generated code and text across multiple languages. We use advanced machine learning techniques to identify subtle patterns that distinguish human work from AI-generated content.

How do you avoid false accusations?

With our industry-low false positive rate of just 0.2%, we minimize incorrect detections. Our system provides confidence scores rather than binary determinations, allowing users to make informed decisions based on the results.
